Felicity is going to be a Hypixel Skyblock replica as an Idle game.

## v1 Plan:

- Basic levelable skills:
    - Combat
    - Farming
    - Mining
- Collections / resources:
    - The more resources you get the more you level up your collection of that item which gives benefits
- Coins (currency system)
- Minions:
    - Passive worker that generates resources, and Skill XP (works offline)
